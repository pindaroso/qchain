* qchain

Questionnaires on the Ethereum blockchain.

** Requirements

- Docker

** Setup

*Development*

=./deployment/bin/compose dev=

*Production*

=./deployment/bin/compose bounce=

Or...

=./deployment/bin/compose build=

followed by:

=./deployment/bin/compose up=

** Deploying

=./deployment/bin/compose deploy <host>=
